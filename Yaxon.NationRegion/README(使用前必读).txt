一、在数据库执行/Other的UCML_NationRegion.sql和UCML_NationRegionExpetion.sql脚本；
二、执行是按照省/市/区县/镇/村的先后顺序(在操作菜单中)；
三、如果中途有关闭程序的话，有两种方案：
	1、去数据库把数据全部删除，
	2、重开始断的地方在数据库把相应的数据删除，
	(比如:在执行层级【镇】中途程序被关闭的话，那就要把整对应的上级【区县】IsLowerUp = 0)
	SQL语句(DELETE FROM dbo.UCML_NationRegion WHERE [Level] =4;UPDATE UCML_NationRegion SET IsLowerUp =0WHERE [Level] =3;)注Level:1=省;2=市;3=区县;4=镇;5=村。
四、当我们把全部数据都执行完后，可以点击其它菜单选项中的异常处理数据(注：由于我们平凡的请求网站可能会造成403，501的状态)；
注：执行的时间大约1.5-2小时。
